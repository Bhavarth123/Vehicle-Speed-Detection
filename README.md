The AI-Based Vehicle Speed Detection and Tracking System for College Campuses is a smart surveillance and safety solution developed using Deep Learning, Computer Vision, Python, OpenCV, and YOLOv8. The main objective of this project is to monitor vehicle movement inside college premises and estimate vehicle speed automatically using CCTV or recorded video footage.
In many college campuses, overspeeding vehicles near academic buildings, parking areas, and student zones can create safety risks. Traditional monitoring methods require manual supervision.This project provides a low-cost and intelligent alternative by using AI-based vehicle detection and tracking.
The system uses the YOLOv8 deep learning model to detect vehicles such as cars, bikes, buses, and trucks. Each vehicle is tracked across video frames, and its speed is estimated based on movement and time difference between frames.

Objectives
To detect vehicles automatically from video footage
To estimate vehicle speed using AI techniques
To improve safety inside college campuses
To create a smart surveillance and monitoring system

Features
Real-time vehicle detection
Speed estimation
Overspeed detection
Smart campus monitoring system
