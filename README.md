Introduction

The AI-Based Vehicle Speed Detection and Tracking System for College Campuses is a smart surveillance and safety solution developed using Deep Learning, Computer Vision, Python, OpenCV, and YOLOv8. The main objective of this project is to monitor vehicle movement inside college premises and automatically estimate vehicle speed using CCTV cameras or recorded video footage.

In many college campuses, overspeeding vehicles near academic buildings, parking areas, hostels, and student zones can create serious safety risks. Traditional monitoring methods require continuous manual supervision and are not always effective. This project provides a low-cost, intelligent, and automated solution for vehicle monitoring and campus safety.

The system uses the YOLOv8 deep learning model to detect vehicles such as cars, bikes, buses, and trucks in real time. Each detected vehicle is tracked across multiple video frames, and its speed is estimated based on movement and time difference between frames. The system can also identify overspeeding vehicles and support smart campus surveillance.

To track moving vehicles in real time

To identify overspeeding vehicles inside campus areas
To improve safety and monitoring in college campuses
